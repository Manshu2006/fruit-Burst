# 🍉 Fruit Burst

Fruit Burst is a hand gesture-controlled fruit Burst game built with Python. Instead of using a mouse, players use their finger in front of a webcam to slice fruits on the screen. The game uses computer vision and AI to detect hand movements in real time.

## 🎮 Features

- ✋ Hand gesture control using webcam
- 🍎 Blast fruits with finger movement
- 💣 Bomb detection (avoid slicing bombs)
- 🎯 Real-time score tracking
- ❤️ Lives system
- 🔊 Sound effects
- 🖥️ Simple and interactive UI

## 🛠️ Technologies Used

- Python
- OpenCV
- MediaPipe
- Pygame
- NumPy

- ## 🎯 How to Play

- Start the game.
- Allow webcam access.
- Move your index finger in front of the camera.
- Blast fruits to earn points.
- Avoid bombs or you will lose a life.
- Try to achieve the highest score.
